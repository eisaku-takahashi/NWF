# NWF AI Workflow Index v2.0.0

---

## 1. 概要

本ドキュメントは Novel Writing Framework（NWF）におけるAIワークフロー仕様の全体構造を定義する。

AI Workflow Spec は、AIを活用した物語制作プロセスの標準化を目的とする。

---

## 2. 位置付け

AI Workflow は以下の基盤仕様の上に構築される。

- Core Spec  
- Architecture Spec  
- Engine Spec  
- Data Spec  
- Execution Spec  

これらの仕様に基づき、AIが構造的に物語制作を支援する。

---

## 3. 適用領域

AIは以下の領域で使用される。

- 物語生成（Story Generation）  
- 物語分析（Story Analysis）  
- 改稿支援（Story Editing）  
- 創作協働（Creative Collaboration）  

---

## 4. 構成ドキュメント

AI Workflow Spec は以下のドキュメントで構成される。

- NWF_AI_Story_Generation_Workflow_v2.0.0.md  
- NWF_AI_Story_Analysis_Workflow_v2.0.0.md  
- NWF_AI_Editing_Workflow_v2.0.0.md  
- NWF_AI_Collaboration_Model_v2.0.0.md  

---

## 5. AIの役割

AIは以下の役割を担う。

- 創作補助  
- 構造分析  
- 改稿支援  
- アイデア生成  

AIはあくまで補助ツールであり、創作主体は人間である。

---

## 6. 設計方針

本仕様は以下の方針に基づく。

- 人間中心設計（Human-in-the-Loop）  
- 構造ベース設計  
- 再現可能なプロセス  
- 透明性のある出力  

---

## 7. まとめ

AI Workflow Spec は、NWFにおけるAI活用の基盤となる仕様である。

本仕様により、人間とAIの協働による体系的な物語制作が可能となる。
