# NWF AI Workflow Index v1.0

## 1. 概要

本ドキュメントは Novel Writing Framework（NWF）における
AIワークフロー仕様を定義する。

AI Workflow Spec は以下の目的で設計されている。

- AIによる物語生成
- AIによる物語分析
- AIによる改稿支援
- 人間とAIの協働

---

## 2. AI Workflow の位置付け

NWFにおけるAI Workflowは次の構造の上に構築される。

Core Spec  
Architecture Spec  
Engine Spec  
Data Spec  
Execution Spec

これらの基盤仕様の上でAIが物語制作を支援する。

---

## 3. AI Workflow の領域

AIは以下の領域で使用される。

Story Generation  
Story Analysis  
Story Editing  
Creative Collaboration

---

## 4. Workflow構成

AI Workflow Spec は以下のドキュメントで構成される。

AI Story Generation Workflow  
AI Story Analysis Workflow  
AI Editing Workflow  
AI Collaboration Model

---

## 5. AIの役割

AIは以下の役割を持つ。

- 創作補助
- 構造分析
- 改稿支援
- アイデア生成

AIは作者の創造性を補助するツールとして設計されている。
