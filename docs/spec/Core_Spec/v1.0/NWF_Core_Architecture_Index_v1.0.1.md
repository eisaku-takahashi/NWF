# NWF Core Architecture Index v1.0.1

## 概要

本ドキュメントは NWF Core Spec のインデックスである。

Core Spec は NWF の基本構造を定義する。

## Core Components

NWF Core は以下の要素で構成される。

Glossary  
Data Model  
Thread Graph Model  
Scene Model  
Beat Model  
File System  
Query Language  
Execution Flow  
Narrative Rendering

## File System

NWF のファイル構造は NWF Project Root を基準として定義される。

Specでは以下の表記を使用する。

NWF Project Root

または

<project_root>/

## Specification Versioning

Spec のバージョン管理ルールは Spec Governance に定義される。

各 Spec ファイルはバージョン付きファイル名で管理される。
