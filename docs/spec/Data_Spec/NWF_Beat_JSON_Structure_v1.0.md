# NWF Beat JSON Structure v1.0

## 1. 概要

Beatは物語の最小イベント単位である。

---

## 2. Beatの役割

Beatは以下を表現する。

- 行動
- 発言
- 情報提示
- 感情変化

---

## 3. Beat基本情報

Beatは以下を持つ。

id  
type  
description

---

## 4. Beatタイプ

Action  
Dialogue  
Thought  
Reveal

---

## 5. Sceneとの関係

複数のBeatが集まりSceneを構成する。

---

v1.0
