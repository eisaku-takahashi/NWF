# NWF CharacterData v2.0.0

---

## 1. 概要

CharacterData は物語内のキャラクターに関する属性・状態・能力・関係性を定義するデータ構造である。

---

## 2. データ構造

- character_id: 一意識別子
- name: 名前
- role: 役割
- attributes: 能力値や性格特徴
- status: 現在の状態
- relationships: 他キャラクターとの関係性

---

## 3. まとめ

CharacterData は物語の登場人物の基本情報と動的状態を管理する中心データである。
