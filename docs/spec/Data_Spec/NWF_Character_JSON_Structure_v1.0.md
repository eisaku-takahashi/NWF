# NWF Character JSON Structure v1.0

## 1. 概要

Characterは物語の登場人物を定義する。

---

## 2. 基本情報

Characterは以下を持つ。

id  
name  
role  
description

---

## 3. キャラクター属性

- 性格
- 動機
- 目標
- 内的葛藤

---

## 4. ストーリーとの関係

Characterは複数のThreadやSceneに関与する。

---

## 5. 成長アーク

Characterは成長曲線を持つ。

---

v1.0
