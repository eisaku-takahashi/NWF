# NWF ConflictData v2.0.0

---

## 1. 概要

ConflictData は物語内の対立や葛藤、衝突状況を管理するデータ構造である。

---

## 2. データ構造

- conflict_id: 一意識別子
- participants: 関与キャラクターリスト
- conflict_type: 対立の種類（例: 権力, 感情, 価値観）
- intensity: 強度 (0.0-1.0)
- status: 進行状態
- resolution: 解決方法または結果

---

## 3. まとめ

ConflictData は物語におけるキャラクター間の葛藤を定義・追跡するためのデータである。
