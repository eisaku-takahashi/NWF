# NWF EmotionalCurve JSON Structure v1.0

## 1. 概要

EmotionalCurveは物語の感情変化を表すデータである。

---

## 2. 目的

物語の感情的なリズムを分析する。

---

## 3. 構造

EmotionalCurveは以下を持つ。

scene_id  
emotion_value  
emotion_type

---

## 4. 感情タイプ

joy  
fear  
sadness  
anger  
hope

---

## 5. Engine連携

EmotionalCurveEngineがこのデータを解析する。

---

## 6. Version

v1.0
