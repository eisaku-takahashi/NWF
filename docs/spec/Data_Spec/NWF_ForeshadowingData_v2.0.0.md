# NWF ForeshadowingData v2.0.0

---

## 1. 概要

ForeshadowingData は未来の状態や出来事への伏線情報を管理するデータ構造である。

---

## 2. データ構造

- foreshadowing_id: 一意識別子
- setup_point: 伏線設置位置
- target_state: 予告対象の状態
- resolution_point: 回収位置
- validity_score: 妥当性 (0.0-1.0)
- context: 関連情報や条件

---

## 3. まとめ

ForeshadowingData は物語における因果的伏線を定義し、回収整合性を検証するためのデータである。
