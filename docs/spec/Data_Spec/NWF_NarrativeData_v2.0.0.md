# NWF NarrativeData v2.0.0

---

## 1. 概要

NarrativeData は物語の語り構造、視点、情報開示制御を管理するデータ構造である。

---

## 2. データ構造

- narrative_id: 一意識別子
- pov: 視点
- timeline_mapping: 時系列対応
- disclosure_control: 情報開示割合 (0.0-1.0)
- content_blocks: 表示内容ブロック

---

## 3. まとめ

NarrativeData は物語の情報提示順序と視点管理を定義するデータである。
