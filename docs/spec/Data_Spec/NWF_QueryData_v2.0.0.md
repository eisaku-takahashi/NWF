# NWF QueryData v2.0.0

---

## 1. 概要

QueryData は NWF 内のデータ検索・抽出・分析条件を管理するデータ構造である。

---

## 2. データ構造

- query_id: 一意識別子
- target: 対象エンジンまたはデータ
- condition: 抽出条件式
- output_format: 出力形式
- timestamp: 作成日時

---

## 3. まとめ

QueryData は NWF のデータアクセス条件を定義し、分析・抽出の基盤を提供する。
