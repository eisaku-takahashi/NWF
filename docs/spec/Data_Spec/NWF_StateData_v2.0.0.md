# NWF StateData v2.0.0

---

## 1. 概要

StateData は物語内の状態（State）および状態変化を管理するデータ構造である。

---

## 2. データ構造

- state_id: 一意識別子
- entity: 状態対象（キャラクター・関係性・世界・情報など）
- attributes: 属性値
- current_value: 現在値
- history: 更新履歴
- timestamp: 記録時刻

---

## 3. まとめ

StateData は物語の動的状態と因果関係を追跡・管理する中心データである。
