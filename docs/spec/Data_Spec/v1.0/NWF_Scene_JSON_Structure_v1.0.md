# NWF Scene JSON Structure v1.0

## 1. 概要

Sceneは物語の具体的な出来事を表す単位である。

Sceneは複数のBeatで構成される。

---

## 2. Sceneの役割

Sceneは以下の目的を持つ。

- ストーリーの進行
- キャラクターの行動
- 情報提示
- 伏線設置

---

## 3. Scene基本情報

Sceneは以下の情報を持つ。

id  
title  
location  
time  
summary

---

## 4. Scene参加キャラクター

Sceneには登場人物が定義される。

---

## 5. Beat構造

Sceneは複数のBeatで構成される。

Beatは最小単位の物語イベントである。

---

## 6. Scene目的

Sceneは次の目的を持つことができる。

Conflict  
Reveal  
Setup  
Resolution

---

v1.0
