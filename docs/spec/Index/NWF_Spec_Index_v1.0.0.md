# NWF Spec Index v1.0

## 概要

このドキュメントは NWF（Novel Writing Framework）の仕様書全体の入口となるインデックスである。

NWFは物語設計・物語データ管理・AI支援執筆を目的としたフレームワークであり、多数の仕様ドキュメントによって構成されている。

本インデックスは以下を目的とする。

- NWF仕様全体の構造を明確にする
- 各仕様ドキュメントへの導線を提供する
- フレームワーク理解の順序を示す


## NWF Spec 全体構造

NWFの仕様は大きく以下のカテゴリで構成される。

Core Spec  
Architecture Spec  
Engine Spec  
AI Interface

NWF仕様を理解するための推奨読書順は以下である。
